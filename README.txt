configuracion del Backend:
 1.-intalar dependencias:
  *asegurate de tener NODE.js instalado.En el caso de que no sea asi, puedes descargarlo en su pagina oficial.
  *abre la terminal y navega a la carpeta del proyecto.
  *ejecuta el siguiente comando dpara instalar las dependendencias necesarias:
  npm install express sqlite3 body-parser cors
2.-ejecutar el servidor
  *hacer un archivo javascript llamado:server.js(sera nuestro servidor)
  *en la terminal,  se le colocara donde se encuentra el archivo server.js y ejecutaras el siguiente comando:
  node server.js
